# PRD 04 — Generador de Música Retro (Tracker)

## 1. Overview
Aplicación para crear música retro basada en patrones y canales limitados.

## 2. Objetivo del Producto
Generar música compatible con juegos de estilo retro.

## 3. Usuarios Objetivo
- Desarrolladores de juegos retro
- Músicos chiptune

## 4. Casos de Uso Principales
1. Crear patrones musicales
2. Componer una canción
3. Reproducir en tiempo real
4. Exportar resultados

## 5. Alcance (In-Scope)
- Editor por patrones
- Canales limitados
- Síntesis simple

## 6. Fuera de Alcance (Out-of-Scope)
- Audio de alta fidelidad
- Mezcla avanzada

## 7. Requisitos Funcionales
1. Edición de patrones
2. Reproducción en tiempo real
3. Exportación de audio y datos

## 8. Requisitos No Funcionales
- Audio sin glitches

## 9. Inputs y Outputs
### Inputs
- Notas y patrones

### Outputs
- Archivo de audio
- Datos de canción

## 10. Flujos Clave
Componer → Organizar → Exportar

## 11. Edge Cases Conocidos
- Patrones vacíos
- Bucles infinitos

## 12. Métricas de Éxito
- Audio exportado reproducible

## 13. Suposiciones y Restricciones
- BPM fijo

## 14. Riesgos
- Limitaciones de síntesis
